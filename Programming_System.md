### Python

```python
import math
# round()并非向下取整，四舍六入五成双, 负数同理
print(round(4.5))
print(round(5.5))
print(round(-2.1))
print(round(-2.6))
print(round(-2.5))
# floor()才是向下取整
print(math.floor(2.5))
print(math.floor(2.9))
print(math.floor(-2.1))
print(math.floor(-1.9))
```
